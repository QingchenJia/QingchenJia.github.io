# Vue3中使用ECharts创建常用统计图表文章设计

## 目标

新增一篇面向已有 Vue 3 基础的前端开发者教程，说明如何在 Vue 3 组件中使用 ECharts 创建常用统计图表。文章跳过安装过程，直接从可复用的组件写法与图表配置开始，使读者能够将示例代码替换到现有项目中使用。

## 文章元数据

- 标题：`Vue3中使用ECharts创建常用统计图表`
- 分类：`前端可视化`
- 标签：`Vue3`、`ECharts`、`数据可视化`、`前端开发`
- 新增文件：`source/_posts/Vue3中使用ECharts创建常用统计图表.md`

## 范围

文章包含以下内容：

1. Vue 3 `<script setup>` 中的基础图表组件模板，说明图表容器、`echarts.init`、`setOption`、`resize` 与 `dispose` 的使用。
2. ECharts 常见配置对象概览，包括 `title`、`tooltip`、`legend`、`grid`、坐标轴、`series` 与 `dataset`。
3. 折线图、柱状图、饼图、散点图、雷达图、漏斗图、仪表盘和热力图八类案例。
4. 柱状图与折线图结合的双 Y 轴组合图案例。
5. 接口数据更新、窗口尺寸变化和容器高度等常见实践注意事项。

文章不包含以下内容：

- ECharts、Vue 或构建工具的安装步骤；
- Vue 2、React 或原生 HTML 的并行实现；
- 地图、K 线图、关系图、树图等需要额外篇幅或资源的图表；
- 可下载的示例工程或新增图片资源。

## 内容组织

文章采用“共用组件模板 + 可替换 option 案例”的方式组织：

1. 前言明确前置条件：读者已经有 Vue 3 项目，并已可导入 `echarts`。
2. 先给出一个可运行的 `.vue` 组件骨架，后续图表只需要替换数据与 `option`。
3. 各图表案例均包含适用场景、示例数据、配置代码和关键效果说明，避免只列 API 配置。
4. 组合图章节说明不同量纲需要使用不同 Y 轴。
5. 结尾说明在真实页面接入异步数据时的更新与资源释放方式。

## 示例约束

- 所有完整组件示例使用 Vue 3 Composition API 与 `<script setup>`。
- 图表配置保持能直接粘贴至组件中的 JavaScript 写法。
- 业务数据采用销售额、访问量、渠道占比、产品能力评分等易理解的演示数据。
- 数据为静态示例，不依赖接口、地图注册或其他外部资源。
- 热力图使用二维分类坐标数据，避免引入地图依赖。
